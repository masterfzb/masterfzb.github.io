项目概述

基于八字命理的紫微斗数自动排盘工具。用户输入出生年月日时，系统自动计算命宫、十二宫位分布及星曜排列，生成完整的紫微斗数命盘。

项目的核心价值在于：将传统命理学的复杂排盘规则编码为可计算的算法，并结合高精度排盘引擎进行验证，探索传统文化与 AI 辅助分析的交叉点。

技术实现

  • 八字计算：根据公历日期与时辰换算天干地支，确定四柱八字
  • 命宫定位：基于生月与生时推算命宫地支，映射到十二宫方位
  • 十二宫排布：从命宫起，逆时针排布十二人事宫位
  • 星曜安放：根据紫微斗数安星法则，计算紫微星系与天府星系各星位置
  • 数据验证：与高精度排盘引擎（v587）交叉验证结果准确性

后续方向

  • 命盘解读 LLM 集成：将排盘结果输入大模型，生成通俗易懂的命理分析
  • 可视化图表：生成十二宫星曜分布的 SVG/Canvas 图表
  • Web API 化：提供在线排盘接口服务
