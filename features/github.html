GitHub 探索

热搜仓库 · 趋势分析 · 一键克隆到 Gitea
GitHub API Gitea Playwright 截图预览

在 MessageCage 内直接浏览 GitHub 热门仓库，查看 README、文件树，一键克隆到本地 Gitea。无需离开应用即可追踪开源动态。

核心功能

  • 热搜追踪：实时 GitHub Trending，按语言/时间筛选
  • 仓库详情：README 渲染、文件树浏览、Star 历史
  • 一键克隆：直接克隆目标仓库到本地 Gitea，进度实时显示
  • 搜索：GitHub 仓库全文搜索
  • 截图预览：Playwright 自动化截取仓库首页
